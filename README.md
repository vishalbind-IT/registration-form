# Student Registration Form

This project contains a modern and responsive **Student Registration Form** built using **HTML and CSS only**.  
The design includes gradient backgrounds, structured input fields, hobby selections, qualification tables, and clean form actions.

### Features
- Fully responsive form layout  
- Gradient header design  
- Input fields for personal, academic, and contact details  
- Radio buttons and checkboxes  
- Qualification table section  
- Reset and Submit buttons  
- Pure **HTML + CSS**

### How to Use
Simply open the `registrationform.html` file in any modern web browser to view the form.
